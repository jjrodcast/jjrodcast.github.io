AlloyExpandableTextView

@Composable()
fun AlloyExpandableTextView(isExpanded: Boolean, text: String, textShowMore: String, textShowLess: String, modifier: Modifier = Modifier, textColor: AlloyStateColors = AlloyTextColor.Primary, textStyle: TextStyle = AlloyTextStyle.Body, expandableTextColor: Color = AlloyTheme.colors.primary, maxLines: Int = Int.MAX_VALUE, enabled: Boolean = true, onTextExpanded: (Boolean) -> Unit)

AlloyExpandableTextView displays part of the text based on the maxLines value. You can expand or collapse the text. Configuration is also avaibla using AlloyTextStyle and AlloyTextColor. By default it uses the animateContentSize animation.

Parameters

isExpanded

Indicates if the text is expanded or collapse by default.

text

The text to be displayed.

textShowMore

Text to show at the end of the paragraph when it's collapsed.

textShowLess

Text to show at the end of the paragraph when it's expanded.

modifier

Modifier to apply to the AlloyExpandableTextView.

textColor

Color to be applied to the AlloyExpandableTextView. See AlloyTextColor.

textStyle

Style configuration for the text such as color, font, line height etc.

expandableTextColor

Color for the expandable/clickable text (textShowMore, textShowLess).

maxLines

An optional maximum number of lines for the text to span, wrapping if necessary. If the text exceeds the given number of lines, it will be truncated according to using the TextOverflow.Ellipsis value.

enabled

Whether the component is enabled or disabled.

onTextExpanded

Callback that is executed when the text is expanded or collapsed. It returns the new state for the isExpanded parameter.
