AlloyTagGroup display a group of AlloyTag.AlloyTag components in a multiline view. The allowed items are AlloyTagText and AlloyTagIcon.

Parameters

tags

List of AlloyTagItem objects that represent each AlloyTag.AlloyTag component.

onClick

Callback that is executed when the user interacts with the children components. It returns the index of the clicked component.

modifier

Modifier to apply to the AlloyTagGroup.

onIconClick

Callback that is executed when the user interacts with the icon of the children components. It returns the index of the clicked component.
